css 선택자란?

특정한 html 태그를 선택하여 스타일을 꾸밀때 활용
선택자 종류 : 태그 #id .class 후손 자신 이벤트

1. 모든 선택자

html 문서 안에 모든 태그를 선택하는 기능

2. 태그 선택자

html 문서만 특정 태그로 설정하는 방법
    ul 태그입니다
  • li 태그입니다

    • 3. 아이디 선택자 #

    html 태그의 속성값인 id을 통해 지정하는 방법
    id의 sytle을 1:1맵핑 될때만 활용 가능
    id는 일반적으로 자바스크립트에서 사용되는 식별자로 가급적 css에서는 사용을 안함
    id1 영역
    id2 영역

    4. 클래스 선택자

    html 태그 속성 중 class로 지정하여 스타일을 적용하는 방법
    class와 sytle를 1:n을 매핑이 가능하여 가장 자주 활용하는 방법
    class1
    class2
